Consultar ofertas realizadas

Consulta la información de todas las ofertas de intercambio de juegos que ha realizado el usuario del token enviado junto con la solicitud.

Petición HTTP#

GET https://api.purnito.codes/api/ofertas/realizadas

Parámetros de la petición.#

No lleva ningún parámetro en el cuerpo de la petición.

Estructura de la respuesta#

La información viene como una arreglo de objetos json, donde cada objeto contiene los datos de una de las ofertas realizadas.

AtributoTipo de datoDescripción
idEnteroIdentificador único de la oferta
RecipienteIDEnteroIdentificador único del usuario que recibe la oferta
OfertanteEnteroIdentificador único del usuario que realiza la oferta
TRStringTítulo del juego que pide el usuario que realiza la oferta
TOStringTítulo del juego que ofrece el usuario que realiza la oferta
nombreStringNombre del usuario que realiza la oferta
estadoStringUna cadena indicando el estatus de la oferta
ApellidoStringApellido del usuario que realiza la oferta

Ejemplo de petición y respuesta#

Ejemplo de petición
curl "https://gameflake.game/api/ofertas/realizadas" \
-X "GET" \
-H "Authorization: Bearer <api_token_aqui>" \
Ejemplo de respuesta
[
{
"id": 4,
"RecipienteID": 4,
"Ofertante": 4,
"TR": "Overcooked 2",
"TO": "Little Nightmares II",
"nombre": "Julian",
"estado": "Pendiente",
"Apellido": "Pedrin"
}
]
